# 🎯 Footfall Detection & Re-ID

AI-powered people counting and re-identification system.

**Stack:** YOLOv8 · BoT-SORT · OpenCV · Streamlit

## Setup

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How to Use

1. Open the app in your browser
2. Upload a video file (mp4, avi, mov, mkv)
3. Adjust settings in the sidebar if needed
4. Click **Run Detection**
5. View results, download annotated video and CSV
